# js-audio-plugin
纯js实现浏览器端录音。

![](https://travis-ci.org/2fps/recorder.svg?branch=master) ![](https://img.shields.io/npm/v/js-audio-recorder.svg) ![](https://img.shields.io/npm/dw/js-audio-recorder.svg)

## 支持功能
+ 支持录音，暂停，恢复，和录音播放。
+ 支持音频数据的压缩，支持单双通道录音。
+ 支持录音时长、录音大小的显示。
+ 支持边录边转（播放）。
+ 支持导出录音文件，格式为pcm或wav。
+ 支持录音波形显示，可自己定制。
